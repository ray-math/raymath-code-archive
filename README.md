# RayMath Code Archive

`RayMath Code Archive`는 Ray수학의 수학 시각화 코드, Python 실험 코드, YouTube 프로젝트 자료,
블로그 관련 산출물, 레거시 스크립트를 모아 둔 공개 아카이브입니다.

이 저장소는 특정 한 도구만을 위한 저장소가 아니라,
수학 콘텐츠 제작 과정에서 나온 다양한 코드와 자료를 장기 보관하고 다시 찾기 쉽게 정리하는 데 목적이 있습니다.

## Ray수학 링크

- YouTube: https://www.youtube.com/@Ray수학
- Blog: https://raymath.pages.dev

## 이 저장소에 들어 있는 것

- 수학 시각화 코드와 애니메이션용 스크립트
- Python 기반 실험 코드와 계산 코드
- YouTube 영상 프로젝트 자료와 관련 소스
- 블로그 글과 연결되는 예제 코드
- 이전 프로젝트에서 남긴 레거시 파일, 메모, 산출물

## 폴더 구조

- 최신 프로젝트는 `2026/03/...` 처럼 `연도/월/프로젝트폴더` 구조로 정리되어 있습니다.
- 오래된 루트 폴더는 각 폴더가 git 이력에 처음 등장한 시점을 기준으로 `연도/월` 아래로 이동했습니다.
- 저장소 루트는 가능한 한 단순하게 유지하고, 실제 자료는 연도별 폴더 아래에 모아 둡니다.

## 자료를 활용하는 방법

1. 관심 있는 주제를 `연도/월` 폴더 아래에서 찾습니다.
2. 프로젝트 폴더 안의 코드, 문서, 씬 파일, 스크립트를 바로 참고합니다.
3. 영상이나 글과 함께 보려면 YouTube 채널과 블로그를 같이 확인합니다.
4. 일부 프로젝트는 Manim을 사용하지만, 저장소 전체는 Manim 전용 아카이브가 아니라 더 넓은 코드 아카이브입니다.

## 어떤 사람에게 유용한가

- 수학 시각화와 교육용 애니메이션 코드가 궁금한 사람
- Python으로 수학 아이디어를 실험하는 방식을 보고 싶은 사람
- Ray수학 YouTube 영상의 코드 배경 자료를 찾는 사람
- 블로그 글과 연결된 예제 코드를 참고하고 싶은 사람
