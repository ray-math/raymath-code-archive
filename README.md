# RayMath YouTube Manim Archive

이 저장소는 `Ray수학` 유튜브에서 다루는 수학 시각화, 실험 코드, 예전 프로젝트 산출물을
백업 형태로 모아 둔 아카이브입니다.

최신 Manim 기반 프로젝트는 날짜 기준으로 정리되어 있습니다.

- `2026/03/...` 처럼 `연도/월/프로젝트폴더` 구조로 보관됩니다.
- 오래된 레거시 폴더는 각 폴더가 처음 git 이력에 등장한 시점을 기준으로 묶었습니다.
- 이 저장소는 보존과 참고가 목적이라 실행 환경 파일보다 자료 자체를 남기는 쪽에 초점을 둡니다.

## 무엇을 찾을 수 있나

- 영상 제작에 사용한 Manim 씬 코드
- 주제별 실험 스크립트와 예전 아이디어 파일
- 일부 프로젝트의 계획 문서, 메모, 산출물

## 자료 활용 방법

1. 관심 있는 주제를 `연도/월` 아래에서 찾습니다.
2. 각 프로젝트 폴더 안의 `scenes`, 스크립트, 문서 파일을 참고합니다.
3. 최신 제작 방식이나 공용 구성 요소가 필요하면 메인 제작 저장소를 함께 봅니다.

## 라이브러리와 메인 제작 저장소

실제 최신 영상 제작 워크플로우와 공용 라이브러리는 별도 저장소에서 관리합니다.

- 메인 저장소: https://github.com/ray-math/manim
- 공용 라이브러리 예시:
  - `scene_library/`
  - `scene_library/style/`
  - `scene_library/templates/`
  - `tooling/`

즉, 이 저장소는 결과물과 기록을 모아 둔 아카이브에 가깝고,
재사용 가능한 제작 기반은 메인 저장소에서 계속 발전합니다.

## Ray수학 채널

- 홈페이지: https://raymath.pages.dev
- 채널 성격: 수학 개념을 시각적으로 설명하고, 아이디어를 코드와 애니메이션으로 풀어내는 프로젝트 아카이브

영상에서 본 코드나 관련 자료를 따라가고 싶다면,
이 저장소와 메인 Manim 저장소를 같이 보는 것이 가장 편합니다.
